# 实验记录索引

本目录记录影响模型结构、输入、训练策略或评估流程的实验动机、实现和结果。

## 记录原则

- 每个实验一个文件，文件名使用 `YYYY-MM-DD-<topic>.md`。
- 训练前先填写动机、假设、改动与训练设置。
- 训练后补充结果、失败案例与结论；不要只记录成功实验。
- 每次比较必须列出基线 checkpoint、数据集划分、评估脚本和阈值。
